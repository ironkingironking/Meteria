# Roadmap

## Near-term

- Tiered tariff pricing model implementation
- Billing job queue with retries and idempotent processing
- Better export customization (per-tenant templates)
- Improved gateway lifecycle and fleet inventory

## Protocol ingestion expansion

- Wireless M-Bus
- LoRaWAN
- Modbus
- M-Bus
- BACnet
- MQTT

## Billing and allocation expansion

- Tenant allocation keys
- Swiss ancillary cost allocation workflows
- DACH billing rule packs
- Final invoice issuance flow
- Credit notes and correction cycles

## Intelligence and operations

- Anomaly detection (consumption spikes, sensor drift, meter reset)
- Forecasting and baseline comparison
- Device health scoring

## Integrations

- ERP integrations: Abacus, Bexio, Swiss21
- Accounting export pipelines
- Webhooks and API event subscriptions

## Product and UI

- Multilingual UI (DE/EN)
- White-label support
- Fine-grained role permissions and SSO
